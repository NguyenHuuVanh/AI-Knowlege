# 🖥️ Hướng Dẫn Dọn Rác Windows 10/11

> Làm theo các bước này định kỳ (1-2 tuần/lần) để máy tính chạy nhanh và mượt hơn.

---

## ✅ Bước 1: Xóa File Tạm Thủ Công

1. Nhấn `Win + R` → gõ **`%temp%`** → Enter
2. Chọn tất cả (`Ctrl + A`) → Xóa (`Delete`)
3. Nhấn `Win + R` → gõ **`temp`** → Enter → xóa tất cả
4. Nhấn `Win + R` → gõ **`prefetch`** → Enter → xóa tất cả

> 💡 Nếu có file không xóa được, hãy bỏ qua (Skip) — đó là file đang được dùng.

---

## ✅ Bước 2: Disk Cleanup (Dọn Đĩa Tích Hợp Sẵn)

1. Nhấn `Win + S` → tìm **"Disk Cleanup"** → mở lên
2. Chọn ổ **C:** → OK
3. Tick chọn **tất cả** các mục
4. Nhấn **"Clean up system files"** → chọn lại tất cả → OK → Delete Files

---

## ✅ Bước 3: Bật Storage Sense (Tự Động Dọn)

1. Vào **Settings** (`Win + I`) → **System** → **Storage**
2. Bật **Storage Sense**
3. Nhấn vào để cấu hình: đặt lịch **Every week** hoặc **Every month**

---

## ✅ Bước 4: Tắt App Khởi Động Cùng Máy

1. Nhấn `Ctrl + Shift + Esc` → tab **Startup**
2. Xem cột **Startup impact**
3. Click chuột phải vào các app **không cần thiết** → **Disable**

> ⚠️ Giữ lại: antivirus, driver âm thanh/đồ họa. Tắt: Spotify, Discord, OneDrive, Zoom... nếu không dùng thường xuyên.

---

## ✅ Bước 5: Xóa Cache Trình Duyệt

### Chrome / Edge:
1. Nhấn `Ctrl + Shift + Delete`
2. Chọn thời gian: **All time**
3. Tick: **Cached images and files**, **Cookies**, **Browsing history**
4. Nhấn **Clear data**

---

## ✅ Bước 6: Gỡ Ứng Dụng Không Dùng

1. Vào **Settings** → **Apps** → **Installed apps**
2. Sắp xếp theo **Size** (lớn → nhỏ)
3. Gỡ các app không dùng đến

---

## ✅ Bước 7: Tối Ưu Hóa Ổ Đĩa (HDD)

> ⚠️ Chỉ làm với ổ **HDD**, **KHÔNG** làm với SSD.

1. Nhấn `Win + S` → tìm **"Defragment and Optimize Drives"**
2. Chọn ổ C: → nhấn **Optimize**

## 📅 Lịch Bảo Trì Gợi Ý

| Tần suất | Việc cần làm |
|---|---|
| **Hàng tuần** | Xóa %temp%, xóa cache trình duyệt |
| **Hàng tháng** | Disk Cleanup, gỡ app không dùng |
| **Mỗi quý** | Defrag HDD, kiểm tra & cập nhật driver |
