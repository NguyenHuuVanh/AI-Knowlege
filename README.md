# WORK

Thư mục làm việc để lưu những tài liệu khi làm việc ở các công ty.

## Cấu trúc thư mục

```
WORK/
├── README.md
├── ai-prompts/          # Thư mục lưu các prompt chuẩn, cấu trúc AI
├── companies/           # Tài liệu theo từng công ty
│   └── <tên-công-ty>/
│       ├── contracts/   # Hợp đồng lao động, phụ lục
│       ├── documents/   # Tài liệu nội bộ, quy trình, hướng dẫn
│       ├── projects/    # Dự án đã tham gia
│       └── notes/       # Ghi chú cá nhân
└── tools/               # Tài liệu về các công cụ, hệ thống, frameworks, AI
    └── <tên-công-cụ>/   # Ví dụ: openclaw
```

## Hướng dẫn sử dụng

1. Tạo thư mục mới trong `companies/` với tên công ty tương ứng.
2. Lưu tài liệu liên quan đến công ty vào các thư mục con phù hợp (contracts, documents, projects, notes).
3. Đặt các tài liệu hướng dẫn về prompt AI tại `ai-prompts/`.
4. Ghi chép, tài liệu nghiên cứu về các phần mềm, công cụ, framework độc lập (như OpenClaw) vào thư mục `tools/<tên-công-cụ>/`.
