# ✍️ Prompt Engineering

## Tổng quan
Nghệ thuật và kỹ thuật viết prompt hiệu quả để tận dụng tối đa khả năng của LLM.

## Các kỹ thuật chính

### 1. Zero-shot
- Không cung cấp ví dụ
- Dựa vào khả năng sẵn có của model

### 2. Few-shot
- Cung cấp một vài ví dụ mẫu
- Model học pattern từ ví dụ

### 3. Chain-of-Thought (CoT)
- Yêu cầu model suy nghĩ từng bước
- "Let's think step by step"

### 4. Tree of Thoughts (ToT)
- Explore nhiều hướng suy nghĩ
- Chọn hướng tốt nhất

### 5. ReAct (Reasoning + Acting)
- Kết hợp reasoning với action
- Nền tảng cho AI Agents

### 6. System Prompts
- Định hình behavior của model
- Role, constraints, output format

## Best Practices
- Rõ ràng và cụ thể
- Cung cấp context đầy đủ
- Định dạng output mong muốn
- Sử dụng delimiters (```, ---, <>)
- Iterative refinement
