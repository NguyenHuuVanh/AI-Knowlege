# 🚀 MLOps & Deployment

## Tổng quan
Kiến thức về triển khai, vận hành, và monitoring các hệ thống AI trong production.

## Chủ đề chính

### 1. Model Serving
- **vLLM** - High-throughput LLM serving
- **TGI** (Text Generation Inference) - Hugging Face
- **Ollama** - Local model serving
- **TensorRT-LLM** - NVIDIA optimized inference
- **ONNX Runtime** - Cross-platform inference

### 2. Deployment Platforms
- **Hugging Face Spaces** - Demo & hosting
- **Replicate** - API cho ML models
- **Modal** - Serverless GPU
- **RunPod** - GPU cloud
- **AWS SageMaker** - Enterprise ML platform
- **Google Vertex AI** - Google Cloud ML

### 3. Optimization
- Quantization (GPTQ, AWQ, GGUF)
- Batching & Caching
- Model pruning
- Knowledge Distillation

### 4. Monitoring & Observability
- LangSmith (LangChain)
- Langfuse (open-source)
- Weights & Biases
- MLflow
