# 🤖 Large Language Models (LLMs)

## Tổng quan
Kiến thức về các mô hình ngôn ngữ lớn - nền tảng của AI hiện đại.

## Chủ đề chính

### 1. Các mô hình phổ biến
| Mô hình | Nhà phát triển | Đặc điểm |
|---------|---------------|-----------|
| GPT-4o / GPT-4.5 | OpenAI | Multi-modal, reasoning mạnh |
| Claude 3.5 / 4 | Anthropic | An toàn, context dài |
| Gemini 2.0 | Google | Multi-modal, tích hợp Google |
| Llama 3 | Meta | Open-source, community mạnh |
| Mistral / Mixtral | Mistral AI | Efficient, open-source |
| DeepSeek | DeepSeek | Open-source, cost-effective |
| Qwen | Alibaba | Multi-lingual |

### 2. Kiến trúc & Kỹ thuật
- Transformer Architecture (Encoder-Decoder, Decoder-only)
- Tokenization (BPE, SentencePiece, tiktoken)
- Context Window & Positional Encoding
- Mixture of Experts (MoE)
- KV Cache & Inference Optimization

### 3. Khả năng
- Text Generation & Completion
- Code Generation
- Reasoning & Chain-of-Thought
- Multi-modal (Text + Image + Audio + Video)
- Function Calling / Tool Use
- Structured Output (JSON mode)

### 4. APIs & Pricing
- OpenAI API
- Anthropic API
- Google Gemini API
- Together AI / Groq / Fireworks (inference providers)
